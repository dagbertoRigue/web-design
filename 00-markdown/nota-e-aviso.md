Para adicionar marcações de "Nota" e de "Aviso" deve-se usar a sintaxe abaixo:  

Sintaxe:  

\> \*\*Note\*\*: Este texto é referente à uma nota.  

\> \*\*Warning\*\*: Este texto é referente à um aviso.  

Por exemplo:  

> **Note**: Este texto é referente à uma nota.  
> 
> **Warning**: Este texto é referente à um aviso.
