Principais Elementos HTML

<area> - define uma área dentro de um mapa de imagem que possui áreas clicáveis ​​predefinidas. Um mapa de imagem permite que áreas geométricas em uma imagem sejam associadas a links de hipertexto.

Este elemento é usado apenas dentro de um elemento <map>.

Exemplo:

Weather forecast for Seattle

03 March 2018

Rain.

04 March 2018

Periods of rain.

05 March 2018

Heavy rain.
